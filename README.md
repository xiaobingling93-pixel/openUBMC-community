## 如何在本代码仓贡献

+ 如果你想通过本代码仓完成社区维护仓库的新建、修改等操作，请参考[SIG组管理](sigs/README.md)。

+ 当openUBMC社区有信息刷新时，这里的文档也需要同步刷新。很感谢您愿意帮助刷新内容。
  请阅读[如何贡献](/CONTRIBUTING.md)获得帮助。
