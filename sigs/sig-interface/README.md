# sig-interface
openUBMC社区接口管理技术决策组织，负责接口管理流程制定与工具化管理、接口规范制定、技术与架构演进规划、知识库建设等

# 组织会议

例会周期：每月的第1周和第3周的周四下午 14:15-16:30 (如遇节假日则顺延至下一个工作日)<br>
组织方式：maintainer按月轮流主持<br>
例会内容：<br>
1、接口管理技术方案讨论与决策<br>
2、接口设计评审<br>
3、SIG成员审视
