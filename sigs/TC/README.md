# TC
openUBMC技术委员会（Technical Committee，简称TC）是openUBMC社区的技术决策机构，负责社区技术决策。

# 组织会议

- 公开的会议时间：当前TC在北京时间，每双周周三的上午10:00召开公开的例行线上讨论会议。
- 具体会议参会方式，会提前一天发出，会议议题在[[@etherpad.TC](https://etherpad.openubmc.cn/p/TC)]登记。
- TC会议的议题包括但不限于：
    +  讨论和决策社区技术发展方向
    +  决策社区重大技术事项
    +  决策SIG组的成立和撤销，审视和辅导SIG组日常工作，协作SIG组间技术合作
    +  在SIG组内部无法决策，需要上升的技术方案
    +  决策社区组件的生命周期
- 欢迎任何感兴趣的开发者参加旁听
